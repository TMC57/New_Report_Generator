CM2W API Documentation: Total Quantity Report
Introduction
This API provides access to the Total Quantity Report of the facilities, accessible through a specific
organization in the CM2W platform.
Base URL
https://dev.cm2w.net/cm2w-api/v2/api-key-auth
Authentication
Make sure to replace <API Key> in the Authorization header with the actual API key.
headers: {
 'Authorization': 'Bearer <API Key>'
}
Response
All responses from the API are in JSON format.
Endpoint: /total-qty-report
Method: GET
Retrieve the Total Quantity Report grouped by facilities.
